Данный проект создаёт книгу-квест. Её содержимое наполняет ИИ, а человек ([Игрок]) проходит [Историю] книги вместе с ИИ (ведущий, голос за кадром, мысли в голове).
[Игрок] своими [Ответами] (описание действия, выбор из вариантов, описание своего варианта) участвует в развитии [Истории], он принимает [Решения], общается с [Персонажами] истории, изменет [Мир] в рамках своих возможностей.

Задача ИИ (автора [Мира]):
 - давать [Игроку] описание,
 - принимать [Ответы],
 - контролировать правдоподобность [Ответов] [Игрока],
 - дополнять описание [Мира] (промпт для себя же).

Перед началом квеста, ИИ создаёт:
- [Мир] (world, background)
- Предысторию этого [Мира]
- Основные направления развития [Истории].

Работа приложения с точки зрения пользователя будет заключаться ровно в 2 этапах, которые будут повторяться друг за другом пока Квест не закончится
1. Приложение даёт текст для [Игрока],
2. [Игрок] вводит свой текст.

Т.о. это как перекидывание мячика (ИИ-человек): ход ИИ, ход человека.
Задача ИИ делать всё, чтобы в целом это было похоже на чтение книги, где человек принимает участие в развитии сюжета [Истории].

Под капотом приложение должно каждый раз создавать (изменять или дополнять) промпт для следующего шага нейросети. Принимать ответ пользователя ([Игрока]), дополнять промпт, дополнять текст "книги" и опять передавать ход пользователю.

Первый ход - нейросети, она даёт первые абзацы книги.
