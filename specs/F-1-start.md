# Создание Мира.

Создание [Сеттинга Мира] (или просто [Сеттинга])  происходит многоэтапно.
Цель создания [Мира] - дать для нейросети глубоко проработанный промпт для обработки ответов [Игрока].
Причина разбивки на этапы, блоки и части ([Артефакты]) - уменьшение галлюцинаций и зацикленности.
Любой [Атрефакт] можно будет оценить с помощью другой модели ИИ. 

Главные этапы:

1. Инициализация [Мира],
2. Наполнение [Мира],
3. Дополнение [Мира].

## Инициализация Мира - придумывание основных, коротких описаний (1-3 предложения).
[Инициализиция] [Мира] состоит из пяти блоков:
1. Название [Мира] + описание одним предложением,
2. Описание [Уровней] [Сеттинга],
3. Описание [Популяции] [Сеттинга],
4. Описание [Предыстории] [Сеттинга],
5. Описание [Историй] и [Задач].

## Наполнение Мира - подробное описание и деталицации блоков [Мира].
К каждому блоку даётся развёрнутое литературное описание. Стиль - литературный исследователь даёт описание чего-то, прочитав книгу (?).


### Блок Описания [Уровней] [Сеттинга].
Здесь описываются составные части [Мира] - [Локации] - где [Игрок] может оказаться. Количество уровней [Локаций] должно соответствовать масштабу [Мира]. Если это вселенная (как в звёздных войнах) - 5-6 уровней, если планета - 3-4 уровня, если квартира - достаточно 1-2 уровня. Уровни могут добавляться по ходу [Истории] по необходимости. Каждая составная часть описывается отдельно.
1. Одно Верхнеуровневое Описание всего [Сеттинга] (Например: выдуманная планета или корабль в космосе или только один город). Это описание очень общее, крупными мазками, без детализации. Главное тут, что вся [История] будет происходить исключительно в рамках этого [Сеттинга];
3. Второй уровень [Сеттинга]. Это части общего [Сеттинга], их как правило несколько, но не очень много - 2-5 (Например, континенты Средиземья, комнаты квартиры, Части корабля, Сектора Галактики). Описание в 1-2 предложения;
4. Третий уровень [Сеттинга]. Это составные части Второго уровня. Описание в 1-2 предложения. Общее количество Третьих Уровней должно быть не больше 10 на старте [Истории];
5. Четвертые (и далее уровни) (кроме стартового) генерируются по ходу [Истории];
6. Стартовый уровень - это та [Локация], с которой начинается [История].

### Блок Описания [Популяции] [Сеттинга].
Здесь описывается население [Мира] - с кем [Игрок] может взаимодействовать (общаться, обмениваться, физически контактировать). Каждое население описывается отдельно.
1. Общее описание всего населения вместе. Это вводная часть для дальнейшего составления перечислений (?);
2. Описание [Групп] населения (расы, группировки, народы). Это необходимо только если масштаб [Мира] большой;
3. Описание [Персонажей];
4. Скрытые [Персонажи] или [Группы] - они могут появиться только внезапно, при определённых решениях или действиях [Игрока] (?).


### Блок Описания [Предыстории] [Сеттинга].
Здесь должно быть описание действий, ситуаций, условий для начала Сюжета [Истории] для [Игрока].
Эта информация, дающая ответы на вопросы: "Что сейчас просиходит?", "Почему это происходит?".


### Блок Описания [Историй] и [Задач].
Здесь ИИ планирует, через что он проведёт [Игрока] в текущей "Главе". Какие трудности или приключения будут ждать [Игрока].


## Дополнение Мира
Это регулярный процесс корректировки [Сеттинга] из-за действий [Игрока]. Это нужно, чтобы ИИ держал повествование [Истории] соответствующим всем предыдущим действиям [Игрока].
